Create node server with express and acces to SqlServer
-------------------------------
First create a database 'nodejs', and next table: 

CREATE TABLE [dbo].[_1_Student](
    [Id] [bigint] NULL,
    [Name] [nvarchar](50) NULL,
    [RollNumber] [bigint] NULL
);

Let's initialize new project:
    > npm init

install dependencies: 
    -   express and 
    -   mssql (for all systems) or msnodesqlv8 (native Windows with authentication)
    > npm install express msnodesqlv8 --save

Write code to running server and return the data in json by query: 
    select * from [_1_Student]
for default get endpoint '/'.

Now let's create procedure on sql server by follow:
    CREATE PROCEDURE dbo._1_GetAllStudents 
    AS 
        SELECT Id,Name,RollNumber FROM dbo._1_Student;
    
    EXECUTE dbo._1_GetAllStudents;

Then write a code to return a data from storage procedure for endpoint '/sp': 
    dbo._1_GetAllStudents


Manage Session data:
-------------------------------
Sessions can store their information in different ways. The popular ways to store session data is:
-    In application memory
-    In a cookie
-    In a memory cache
-    In a database

Install  express-session:
    > npm install --save express-session

and write code with 'express-session' to handle the sessions.
